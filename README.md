# Student Management System - Oracle SQL Project

## Author
Fatoumata Bamba KEITA

## Description
This project is a relational database system built using Oracle XE.
It manages students, courses, enrollments, and grades.

## Features
- Relational schema with foreign keys
- Data integrity constraints
- View for student performance
- Automatic grade letter assignment using trigger
- Index optimization on email
- Analytical queries (average score per course)

## Technologies
- Oracle XE
- Oracle SQL Developer
- SQL
- PL/SQL (Trigger)

## How to Run
1. Execute schema.sql
2. Execute data.sql
3. Execute analytics.sql
